# Gamevine.ai — Product Specs

This folder holds the **product specification set** for Gamevine.ai: what the platform is, who it serves, and how the main product areas fit together.

Read **`product-overview.md`** first for the north-star summary and cross-cutting scope. Other documents split the product into areas that can be refined independently before work is broken into Linear epics and tickets.

- Specs describe **product intent**, not implementation or API contracts.
- **Suggested Epics** / **Suggested Tickets** are placeholders for later breakdown into Linear work.
